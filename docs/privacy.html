Política de Privacidade

Última atualização: 23 de junho de 2026

Este aplicativo é uma ferramenta pessoal de automação criada para publicar vídeos curtos (formato vertical) na conta de TikTok do próprio operador da ferramenta. Ele não é oferecido a terceiros e não coleta dados de usuários externos.

1. Quais dados acessamos

Por meio das APIs oficiais do TikTok (Login Kit e Content Posting API), o aplicativo acessa apenas o necessário para publicar vídeos na conta autorizada:

  • Um token de acesso emitido pelo TikTok após você autorizar o aplicativo.
  • Informações básicas da conta de criador necessárias para a publicação (ex.: opções de privacidade permitidas, retornadas pelo endpoint creator_info).
  • O arquivo de vídeo e seus metadados (título/legenda) que você mesmo gerou e optou por publicar.

2. O que NÃO fazemos

  • Não coletamos, vendemos ou compartilhamos dados pessoais com terceiros.
  • Não acessamos mensagens, contatos, lista de seguidores ou conteúdo de outras contas.
  • Não exibimos anúncios nem fazemos rastreamento.

3. Armazenamento e segurança

O token de acesso é armazenado de forma privada (como segredo criptografado no ambiente de automação) e usado exclusivamente para enviar vídeos à conta autorizada. Ele nunca é exposto publicamente. Você pode revogar o acesso a qualquer momento nas configurações de segurança da sua conta TikTok, o que invalida o token imediatamente.

4. Retenção de dados

Não mantemos banco de dados de usuários. Os vídeos publicados passam a ser regidos pelos próprios termos e pela política de privacidade do TikTok.

5. Serviços de terceiros

A publicação é feita via APIs oficiais do TikTok. O uso do TikTok é regido pela Política de Privacidade do TikTok.
