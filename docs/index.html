Tudo é Curioso Vídeos

Ferramenta pessoal de automação que publica vídeos curtos verticais no canal @tudoecuriosovideos, usando as APIs oficiais do TikTok (Login Kit e Content Posting API).

O que esta ferramenta faz

  • Gera vídeos curtos de curiosidades a partir de roteiros próprios.
  • Publica automaticamente, de forma agendada, na conta do próprio operador.
  • Não coleta dados de terceiros nem oferece serviço a outros usuários.
